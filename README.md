🎙️ Voice-2-Note

Voice-2-Note is a React-based web application that allows users to create, edit, and manage notes using **voice input**. It leverages the Web Speech API to convert speech into text, making note-taking faster and hands-free.

---

✨ Features

- 🎤 Speech-to-text note creation
- 📝 Create, edit, delete notes
- 🔍 Search notes in real time
- 🌓 Light / Dark mode
- 📐 Grid & List view toggle
- 💾 Local storage persistence
- ⚡ Fast and responsive UI

---

🛠️ Tech Stack

- **Frontend:** React + Vite
- **Styling:** CSS
- **Speech Recognition:** Web Speech API
- **State Management:** React Hooks
- **Storage:** LocalStorage
